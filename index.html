โปรเจกต์: Interactive Raster Graphics

นี่คือผลงานการสร้างระบบวาดภาพและเรนเดอร์กราฟิกแบบ Raster (พิกเซล) ด้วย JavaScript บริสุทธิ์บน HTML Canvas โดยไม่ได้ใช้ไลบรารีภายนอก ภายในโปรเจกต์ประกอบด้วยระบบการวาดเส้น, การลงสี, และการแสดงผล Sprite (รูปดอกไม้) ที่สามารถหมุนและซูมได้

วิธีเล่น: ลองใช้เมาส์คลิกลากเพื่อวาดเส้น หรือใช้ ปุ่มลูกศร บนคีย์บอร์ดเพื่อหมุนและซูมดอกไม้ (กดเลข 1, 2, 3 เพื่อเปลี่ยนสีเส้น)
